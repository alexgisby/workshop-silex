# Simple Silex App

To be used for workshops etc.

## Installation:

```sh
$ git clone git@github.com:alexgisby/workshop-silex.git
$ cd workshop-silex
$ make install
```

## Provided

- Silex
- Twig
- A simple route (/hello)
- A namespace for additional classes (DemoApplication -> src/)
- A simple server (`$ ./script/server` or `$ make server`)
- PHPUnit (`$ make test`)
